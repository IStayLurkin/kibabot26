# Kiba Bot

Discord bot with:

- prefix commands only
- expense tracking
- SQLite + aiosqlite
- modular cogs/services/database architecture
- AI chat with memory and summaries
- OpenAI / Hugging Face / Ollama support
- live runtime date and time handling
